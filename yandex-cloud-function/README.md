# Прокси для Yandex Cloud Functions

Точка входа: `index.handler`. Функция принимает POST-запросы конструктора,
последовательно пересылает их в официальный UN Comtrade API и возвращает CORS
для GitHub Pages. GET-запрос используется для проверки доступности.

Рекомендуемые настройки: Node.js 22, память 256 МБ, тайм-аут 60 секунд,
публичный доступ к функции.
